## Crear un mapeo que nos permita convertir el dataset IMDB a revisiones que podamos leer

Las revisiones en el dataset IMDB est√°n codificadas como una secuencia de enteros. Afortunadamente, el dataset tambi√©n contiene un √≠ndice que nos permite volver a una representaci√≥n tipo texto.


```python
# Obtener el √≠ndice de palabras del dataset
word_index = tf.keras.datasets.imdb.get_word_index()

# Asegurarnos de que las palabras "especiales" pueden leerse correctamente
word_index = {k:(v+3) for k,v in word_index.items()}
word_index["<PAD>"] = 0
word_index["<START>"] = 1
word_index["<UNKNOWN>"] = 2
word_index["<UNUSED>"] = 3

# Buscar las palabras en el √≠ndice y hacer una funci√≥n que decodifique cada review
# Si la palabra no est√° devolver√° '?'
reverse_word_index = dict([(value, key) for (key, value) in word_index.items()])
def decode_review(text):
    return ' '.join([reverse_word_index.get(i, '?') for i in text])
```

## Crear y construir una red LSTM recurrente


```python
# Empezamos definiendo una pila vac√≠a. Usaremos esta pila para ir construyendo
# la red, capa por capa
model = tf.keras.models.Sequential()

# La capa de tipo Embedding proporciona un mapeo (tambi√©n llamado Word Embedding)
# para todas las palabras de nuestro conjunto de entrenamiento. En este embedding,
# las palabras que est√°n cerca unas de otras comparten informaci√≥n de contexto
# y/o de significado. Esta transformaci√≥n es aprendida durante el entrenamiento
model.add(
    tf.keras.layers.Embedding(
        input_dim = vocab_size, # Tama√±o del vocabulario
        output_dim = 32, # Dimensionalidad del embedding
        input_length = review_length # Longitud de las secuencias de entrada
    )
)

# Las capas de tipo Dropout combaten el sobre aprendizaje y fuerzan a que el
# modelo aprenda m√∫ltiples representaciones de los mismos datos, ya que pone a
# cero de forma aleatoria algunas de las neuronas durante la fase de
# entrenamiento
model.add(
    tf.keras.layers.Dropout(
        rate=0.25 # Poner a cero aleatoriamente un 25% de las neuronas
    )
)

# Aqu√≠ es donde viene realmente la capa LSTM. Esta capa va a mirar cada
# secuencia de palabras de la revisi√≥n junto con sus embeddings y utilizar√°
# ambos elementos para determinar el sentimiento de la revisi√≥n
model.add(
    tf.keras.layers.LSTM(
        units=32 # La capa va a tener 32 neuronas de tipo LSTM
    )
)

# A√±adir una segunda capa Dropout con el mismo objetivo que la primera
model.add(
    tf.keras.layers.Dropout(
        rate=0.25 # Poner a cero aleatoriamente un 25% de las neuronas
    )
)

# Todas las neuronas LSTM se conectan a un solo nodo en la capa densa. La
# funci√≥n sigmoide determina la salida de este nodo, con un valor entre 0 y 1.
# Cuanto m√°s cercano sea a 1, m√°s positiva es la revisi√≥n
model.add(
    tf.keras.layers.Dense(
        units=1, # Una √∫nica salida
        activation='sigmoid' # Funci√≥n de activaci√≥n sigmoide
    )
)

# Compilar el modelo
model.compile(
    loss=tf.keras.losses.binary_crossentropy, # Entrop√≠a cruzada
    optimizer=tf.keras.optimizers.Adam(), # Optimizador Adam
    metrics=['accuracy']) # M√©trica de los informes

# Mostrar un resumen de la estructura del modelo
model.summary()
```

## Redes GRU

Ahora vamos a repetir el entrenamiento pero con las redes GRU. Recuerda que las redes Gated Recurrent Unit (GRU) implementan una simplificaci√≥n de la neurona LSTM basada en reducir el n√∫mero de puertas, par√°metros y estaddos de la misma.


```python
# Pila vac√≠a
model = tf.keras.models.Sequential()

# Embedding
model.add(
    tf.keras.layers.Embedding(
        input_dim = vocab_size, # Tama√±o del vocabulario
        output_dim = 32, # Dimensionalidad del embedding
        input_length = review_length # Longitud de las secuencias de entrada
    )
)

# Primer Dropout
model.add(
    tf.keras.layers.Dropout(
        rate=0.25 # Poner a cero aleatoriamente un 25% de las neuronas
    )
)

# Capa GRU
model.add(
    tf.keras.layers.GRU(
        units=32 # La capa va a tener 32 neuronas de tipo LSTM
    )
)

# Segundo Dropout
model.add(
    tf.keras.layers.Dropout(
        rate=0.25 # Poner a cero aleatoriamente un 25% de las neuronas
    )
)

# Capa densa final
model.add(
    tf.keras.layers.Dense(
        units=1, # Una √∫nica salida
        activation='sigmoid' # Funci√≥n de activaci√≥n sigmoide
    )
)

# Compilar el modelo
model.compile(
    loss=tf.keras.losses.binary_crossentropy, # Entrop√≠a cruzada
    optimizer=tf.keras.optimizers.Adam(), # Optimizador Adam
    metrics=['accuracy']) # M√©trica de los informes

# Mostrar un resumen de la estructura del modelo
model.summary()
```
